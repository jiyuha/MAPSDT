**Usage**

## 
- 기존 단판성 성격이 강한 테트리스에 육성 요소를 가미
- 자신의 계정의 경험치를 올리며 캐릭터 진화하는 재미
- 자신의 컴퓨터가 아닌 곳에서도 로그인 후 게임 가능
- 다양한 테마 선택으로 지루하지 않은 게임

## 게임 실행 방식

  - 현재 repository의 소스 코드를 모두 다운 또는 clone을 통해 local에 저장

  - 추가 모듈에 표기한 모듈들을 다운 받고 run.py에서 실행
